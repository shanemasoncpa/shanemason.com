# Version 2.5 - March 2024

We're excited to announce the release of Gemifi version 2.5! This update brings several highly-requested features and improvements.

## New Features

### Enhanced Reporting
- New customizable report templates
- Improved PDF export quality
- Scheduled report delivery

### New Integrations
- Schwab integration now available
- Improved Plaid connection reliability
- New tax document import

## Performance Improvements
- 40% faster dashboard load times
- Improved mobile responsiveness
- Better handling of large portfolios

## Bug Fixes
- Fixed calculation issues with certain RSU vesting schedules
- Resolved display issues on Safari
- Various UI polish improvements
